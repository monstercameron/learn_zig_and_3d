# Experiments

This folder contains isolated prototype projects that explore ideas without forcing them into the main renderer immediately.

## Current Experiments

- `hotreload_demo/`: DLL hot-reload proof of concept for Zig-native plugin workflows

## Expectations

- Each experiment should build on its own.
- Experiment-specific artifacts and caches should stay inside the experiment folder and out of git.
- Keep experiment docs local to the experiment when setup differs from the main app.

## Local Docs

- `hotreload_demo/README.md`: setup and workflow for the plugin reload experiment
